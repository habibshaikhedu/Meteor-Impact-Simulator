About Meteor Madness

This simulator uses NASA’s open data to model asteroid impacts.

  • NeoWs API: Near-Earth Object data
  • SBDB: NASA JPL Small-Body Data base API
  • NASA EONET: Earth Observatory
  • DART Mission: Deflection reference

Built for NASA Space Apps Challenge 2025
